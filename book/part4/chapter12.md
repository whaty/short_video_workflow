## 二、进阶学习路径

### 阶段1：新手期（0-3个月）

**学习重点**
```
□ 掌握基础操作
□ 理解平台规则
□ 学会内容创作
□ 建立发布习惯
```

**推荐资源**
```
1. 本书（完整学习）
2. 抖音创作者学院（基础教程）
3. 剪映官方教程（制作技巧）
4. 同领域账号（模仿学习）
```

**行动计划**
```
- 每天发布1-2条
- 每周复盘数据
- 每月总结经验
```

**学习重点**
```
□ 提升内容质量
□ 优化数据表现
□ 探索变现方式
□ 建立个人风格
```

**推荐资源**
```
1. 飞瓜数据（分析爆款）
2. 案例拆解（学习方法）
3. 数据分析（优化策略）
4. 变现课程（商业化）
```

**行动计划**
```
- 稳定日更

### 阶段3：成熟期（6-12个月）

## 四、持续学习建议

### 1. 建立学习系统

#### 每日学习
```
时间：30分钟
内容：
- 浏览行业资讯
- 学习1个新技巧
- 分析1个爆款案例
```

#### 每周学习
```
时间：2小时
内容：
- 深度学习1个主题
- 实践新学的方法
- 总结本周经验
```

#### 每月学习
```
时间：1天
内容：
- 阅读1本书
- 参加1次分享
- 复盘月度数据
```

### 2. 建立知识库

#### 工具推荐
```
- Notion（推荐）

```

```

├── 学习笔记/

├── 模板库/
├── 工具清单/
